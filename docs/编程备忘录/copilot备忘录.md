## 快捷键
-  接受建议：`Tab`
-  拒绝建议：`Esc`
-  触发行内Copilot：`Alt/Option + \` （Coplit还没有给出建议或者建议被拒绝了，希望手工触发它提供建议）
-  打开Copilot：`Ctrl + Enter` （会打开一个单独的面板，展示10个建议）
-  下一条建议：`Alt/Option + ]`
-  上一条建议：`Alt/Option + [`
